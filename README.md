## 🔧 **Troubleshooting**

| Problem                        | Solution                                                                 |
|---------------------------------|--------------------------------------------------------------------------|
| DLCs not showing in-game        | Ensure CreamAPI was run as Administrator and restart Steam.              |
| Game crashes after launch       | Use the “Uninstall” option in CreamAPI and reinstall unlockers.         |
| Missing DLL errors              | Reinstall Koaloader via CreamInstaller’s context menu.                   |
| Still doesn’t work              | Re-run CreamAPI and refresh Steam appinfo for TheHunter: Call of the Wild.    |
